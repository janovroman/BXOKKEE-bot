# База знаний BXOKKEE / ВХОККЕЕ

- BXOKKEE / ВХОККЕЕ — магазин и сервис для хоккея и коньков.
- Основные направления: заточка коньков, профилирование, лезвия, консультации по подбору, доставка, UDS, ремонт.

## Профилирование

- Профилирование коньков стоит 1290 ₽.
- Профилирование — важная дополнительная услуга, которую нужно предлагать мягко и без давления.
- При профилировании заточка входит в подарок.
- При подборе новых лезвий не предлагать профилирование в первом ответе.
- Предлагать профилирование позже: после выбора варианта лезвий, вопроса про покупку/наличие, вопроса “что лучше?” или когда клиент близок к покупке.
- Перед допродажей профилирования логично сначала мягко предложить UDS.
- Не обещать точный результат без осмотра лезвий.

## Желоб и скольжение

- Чем больше цифра желоба, тем он более пологий.
- Более пологий желоб даёт лучшее скольжение.
- У более пологого желоба сопротивление меньше.
- Лезвие меньше врезается в лёд.
- Зацеп ниже.
- Устойчивость ниже.
- Чем меньше цифра желоба, тем желоб глубже, зацеп выше, но скольжение хуже.
- Если клиент спрашивает только про желоб или максимальное скольжение, сначала отвечать только про желоб.

## Лезвия

Бренд лезвий писать строго:

- in hockey® Base
- in hockey® Pro

У клиента есть три варианта:

- бюджетные лезвия без бренда;
- in hockey® Base;
- in hockey® Pro.

Клиенту не называть бюджетные лезвия словами “no name”, “китайские”, “дешёвые”.

Если клиент спрашивает, какого бренда бюджетные лезвия, отвечать: Это бюджетные лезвия без бренда. Если нужен брендовый вариант — можно рассмотреть in hockey® Base или in hockey® Pro.

## Размеры и стаканы лезвий

Достаточные данные для первичного подбора:

- Bauer + Edge + размер;
- Bauer + Tuuk Edge + размер;
- Bauer + Fly / PowerFly + размер;
- CCM + XS + размер;
- Bauer + Vertexx + размер.

Если данные достаточны, не просить фото, не спрашивать модель коньков и не уточнять стакан повторно.

Фото или модель просить только если клиент написал только Bauer, только CCM, только размер, не знает стакан, есть сомнение по совместимости или сам просит проверить по фото.

Bauer Tuuk Edge:

- Если клиент написал Bauer Edge, Bauer Tuuk Edge, Tuuk Edge, Edge, тук едж, туук едж или едж, и в контексте речь про лезвия, считать это Bauer Tuuk Edge.
- Если клиент в контексте лезвий написал трёхзначное число от 200 до 330, считать это размером лезвия.

Bauer Fly / PowerFly:

- Если клиент написал Bauer Fly, Bauer PowerFly, Fly или PowerFly с размером, считать это достаточными данными для первичного подбора.
- Не путать Fly/PowerFly с Bauer Tuuk Edge и Bauer Vertexx.

CCM XS:

- CCM + XS + трёхзначный размер = достаточно для подбора лезвий.
- Примеры: CCM XS 271, CCM 271 XS, XS 271, ццм xs 271.
- Считать как CCM XS, размер 271.
- Не спрашивать фото, модель коньков или стакан, если уже есть CCM XS и размер.

Bauer Vertexx:

- Vertexx — это вратарский стакан Bauer.
- Обычно используется на вратарских коньках Bauer.
- Если клиент пишет “вратарские Bauer”, “вратарские коньки Bauer”, “Bauer goalie”, “goalie Bauer”, агент должен понимать, что, скорее всего, речь про Bauer Vertexx.
- Для подбора нужны: Bauer Vertexx + размер лезвия.
- Если клиент написал Bauer Vertexx + размер, этого достаточно для подбора лезвий.
- Если клиент написал только “вратарские Bauer”, нужно уточнить размер на старом лезвии или попросить фото маркировки.
- Не путать Bauer Vertexx с Bauer Tuuk Edge и Bauer Fly/PowerFly.
