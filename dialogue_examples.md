# Диалоговые примеры

### Пример — короткий правильный ответ

Клиент:
мне нужны лезвия Bauer Edge 272

Правильный ответ:
Понял, нужны лезвия Bauer Tuuk Edge 272.

Есть три варианта:
1. Бюджетные лезвия без бренда
2. in hockey® Base
3. in hockey® Pro

Какой вариант рассматриваете — самый доступный или лучше по ресурсу?

Плохой ответ:
Понял, нужны лезвия Bauer Tuuk Edge 272.

Есть три варианта:
1. ...
2. ...
3. ...

Можем сразу подготовить...
При профилировании...
UDS...
и так далее.

Почему плохо:
Слишком длинно. Клиент ещё не выбрал вариант. Допродажу нужно делать позже.

### Пример — CCM XS

Клиент:
нужны лезвия CCM XS 271

Правильный ответ:
Понял, нужны лезвия CCM XS 271.

Есть три варианта:
1. Бюджетные лезвия без бренда
2. in hockey® Base
3. in hockey® Pro

Какой вариант рассматриваете — самый доступный или лучше по ресурсу?

Плохой ответ:
Пришлите фото или модель коньков.

Почему плохо:
Клиент уже дал стакан CCM XS и размер 271. Этого достаточно для подбора.

### Пример — вратарские Bauer

Клиент:
Нужны лезвия на вратарские Bauer

Правильный ответ:
Скорее всего, у вас стакан Bauer Vertexx. Подскажите размер, который указан на старом лезвии, или пришлите фото стакана — там на пяточной области есть цифра. Подберём точнее.

Плохой ответ:
Какая модель коньков?

Почему плохо:
Для подбора лезвий важнее стакан и размер. Вратарские Bauer чаще всего ведут к Bauer Vertexx, поэтому сначала уточняем размер/маркировку.

### Пример — выбор бюджетных после подбора

Клиент:
Бюджетные

Правильный ответ:
Бюджетные лезвия без бренда для Bauer Tuuk Edge 272 — базовая цена 3490 ₽.

Можем зарегистрировать вас в UDS — получите 500 приветственных бонусов и сможете копить бонусы на следующие покупки и услуги.

Плохой ответ:
Бюджетные лезвия стоят 3490 ₽. Ещё сделаем профилирование, заточку, UDS, скидку и всё подготовим.

Почему плохо:
Слишком много допродажи в одном сообщении. После цены сначала мягко предложить UDS, профилирование можно следующим ответом.
